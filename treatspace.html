Treatspace

Overview

  • Type: Healthcare Software Company in Pittsburgh, PA
  • Role: UX/UI Designer Internship
  • Tools: Sketch/Adobe Creative Suite/Zeplin
  • Timeline: 11 weeks, Summer 2019

Treatspace is an award-winning, multi faceted healthcare platform that enables high-performance referral management, powers online patient engagement, and facilitates clinical collaboration.

As a UX/UI designer, following guidelines from the engineering team and my mentor, I created user interface prototype for a front-facing website editing feature that will be used to design and create PracticeBeat websites. I also had opportunity to create user interface prototype new internal dashboard for the Treatspace platform, online testing tool, and icon designs. Special Thanks to my mentor Naveen Kumar.
